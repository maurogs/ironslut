Reading and writing files in Ruby

We can read the contents of a text file as a string easily with Ruby:

file_contents = IO.read("ruby.rb")
puts "The source file contains: #{file_contents}"

Also, it is very easy to write a file:

puts "What's your name?"
name = gets.chomp
IO.write('name.txt', name)
